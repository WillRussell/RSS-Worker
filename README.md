# RSS Worker

Node.js app that turns youtube videos into podcasts.

## Get Started

Running this project locally requires [yt-dlp](https://github.com/yt-dlp/yt-dlp#installation), a `youtube-dl` fork with additional features and fixes.

### `npm install`

## Scripts

### `node worker`
